Participation WebApp (Streamlit)
--------------------------------

How to use:
1. Install requirements once:
   pip install streamlit pandas openpyxl

2. Run the app:
   streamlit run participation_webapp.py

3. A browser will open at http://localhost:8501

Features:
- Manage students (add/remove, reset scores)
- Random mixed-gender group generation
- Random student picker
- Give or subtract points individually or to groups
- Keep history (saved to Excel)
